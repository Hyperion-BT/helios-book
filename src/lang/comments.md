# Comments

Helios comments are C-like. 

## Single-line comments

Single-line comments use two forward slashes (`//`):

```helios
func main(_, _, _) -> Bool {
	// This is a comment.
	true
}
```

## Multi-line comments

Multi-line comments use `/* ... */`:

```helios
func main(_, _, _) -> Bool {
	/*
		This is a multi-line comment.
	*/
	true
}
```
